## Pre-requisites 

- Java 7 
- Maven 3.3 
- Git 

## Build Instructions 

- Open Command prompt / shell
- Got to the directory where pom.xml is found 
- run the following commands 

```
mvn clean 
mvn compile 
mvn package tomcat7: run

```

The application can be accessed by openning http://localhost:8080/login-service
